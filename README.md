# Molvue: React Native App for Molecular Modeling

A React Native application for modeling molecular and ionic structures with five interactive modules.

## Modules

1. **Formal Charge Calculator & Resonance Analyzer**
2. **Bond Type Predictor from Electronegativity**
3. **3D Molecular Geometry Explorer**
4. **Ionic Compound Lattice Energy Estimator**
5. **Hybridization & Bond Angle Identifier**

## Setup Instructions

1. Install dependencies:
```
npm install
```

2. Start the Expo development server:
```
npx expo start
```

3. Run on your device or emulator:
   - Scan the QR code with the Expo Go app (Android) or Camera app (iOS)
   - Press 'a' for Android emulator
   - Press 'i' for iOS simulator

## Technologies Used

- React Native
- Expo
- TypeScript
- Three.js for 3D visualization
- React Navigation

## Project Structure

- `/src/screens`: Contains all module screens
- `/src/components`: Reusable components
- `/src/utils`: Utility functions and helpers
- `/assets`: Images and other static assets
